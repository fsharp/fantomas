# Website info

To run the website:

> dotnet fsdocs watch

To compile the Sass:

> dotnet fsi .\docs\.style\style.fsx

Add `--watch` to the command to automatically recompile Sass.

> dotnet fsi .\docs\.style\style.fsx --watch

To build the website:

> dotnet fsi .\docs\.style\style.fsx
> dotnet fsdocs build
